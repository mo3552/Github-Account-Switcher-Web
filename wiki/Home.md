## 🎯 주요 기능

-   **Git 글로벌 설정 관리**: `user.name`과 `user.email` 설정 조회 및 변경
-   **계정 관리**: 여러 Git 계정을 저장하고 관리
-   **자동 자격 증명 삭제**: 설정 변경 시 GitHub 자격 증명 자동 삭제
-   **단일 인스턴스 실행**: 중복 실행 방지 및 기존 인스턴스 포커스
-   **크로스 플랫폼 지원**: Windows, macOS, Linux 지원
